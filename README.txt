The Solr Security Manager is a user interface designed to help you manage your Solr Authentication and Authorization
definitions. It provides the following features:

   -- Manage Users, Roles and Permissions via a graphical UI. 
   -- Save security.json versions in your history. 
   -- Easily create users and manage a complex permission matrix. 
   -- Validate JSON for syntactical correctness. 
   -- Easily Load and Unload Configurations
